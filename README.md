# Cyber Sentinel 🛡️

**Cyber Sentinel** is an automated, AI-driven security ecosystem designed for network monitoring, threat intelligence gathering, and incident response.

The project integrates advanced tools like **n8n**, **HashiCorp Vault**, **Google Gemini AI**, and multiple **CTI (Cyber Threat Intelligence)** providers to protect home and enterprise networks.

## 🚀 Core Features

* **Automated Threat Analysis**: Real-time IP and FQDN reputation checks using VirusTotal, ThreatFox, and URLHaus.
* **AI Synthesis**: Leveraging Google Gemini to provide human-readable security verdicts and risk scoring.
* **Secure Credential Management**: Integration with HashiCorp Vault for a "zero-secrets" local configuration.
* **Automated Alerting**: Immediate notifications via Email/Gmail for high-risk indicators.
* **Infrastructure as Code**: Deployment managed through Ansible for consistency and security.

## 🛠️ Tech Stack

* **Logic Orchestration:** n8n (AI-Integrated Workflows)

* **Languages:** Python (Main), JavaScript (Data Transformation)
* **Databases:** MySQL (Relational Data), MongoDB (CTI Data Lake)
